2048 Game
About
A console game of 2048 created for a C++ course project at Sofia University (2025/2026).

Features
Board Size: Choose any size from 4x4 to 10x10.

Gameplay: Use W, A, S, D to move and merge tiles.

Leaderboard: Saves the top 5 scores for each board size to a text file.

Design: Clean grid layout using iomanip.
